# Overview
Cluster 3 'Civil Security for Society' is part of Pillar II of [[Horizon Europe]]. It contributes to a safer and more secure Europe, a 'Preparedness Union', and stronger common borders, protecting democracy and putting R&I at the heart of a resilient economy.

# Strategic Role
This cluster equips security practitioners (law enforcement, critical infrastructure operators) with modern tools consistent with EU values, while boosting European competitiveness and promoting innovation procurement. It supports the ProtectEU strategy and the European Preparedness Union Strategy.

# Policy Context
- Addresses terrorism and cybercrime threats
- Supports European Civil Defence Mechanism development
- Strengthens the EU's [[Cybersecurity|cyber resilience]] through advanced protective measures
- Facilitates secure border crossings for people and goods

# Horizon Europe Relevance
- Budget contribution to climate action: 28.6% (EUR 96 million) in the 2026-2027 WP
- Contributes to [[Open Strategic Autonomy]] in security technologies
- Promotes [[Innovation Procurement]] for security solutions

# Evolution Across Work Programmes
The 2026-2027 WP strengthens alignment with the ProtectEU strategy and the European Preparedness Union Strategy, with new emphasis on crisis management and cyber resilience. Focus on international collaboration for cross-border security challenges.

# Implications for Proposal Design
- Security appraisal procedure applies to all proposals
- Restrictions on participation may apply for sensitive topics
